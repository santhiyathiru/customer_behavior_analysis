# 🛍️ Customer Shopping Behavior Analysis

## 📌 Overview

Customer Shopping Behavior Analysis is a data analytics project that analyzes customer purchasing patterns, sales trends, and product performance using retail data. The project demonstrates the complete data analytics workflow, including data loading, cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and business reporting.

The main objective is to transform raw customer data into meaningful insights that help businesses understand customer behavior and improve decision-making.

---

## 📂 Dataset

The dataset contains customer shopping transaction details, including:

- Customer ID
- Age
- Gender
- Product Category
- Item Purchased
- Purchase Amount
- Payment Method
- Purchase Frequency
- Location
- Season
- Discount Applied
- Shipping Type
- Review Rating
- Subscription Status

## 📋 Project Workflow

### 1. Data Loading
- Imported the dataset into Python using Pandas.
- Checked data types and dataset structure.

### 2. Data Cleaning
- Removed duplicate records.
- Handled missing values.
- Corrected inconsistent data.
- Converted data into appropriate formats.

### 3. Exploratory Data Analysis (EDA)
Performed analysis to answer questions such as:
- Which product categories generate the highest sales?
- Which gender purchases more?
- What is the average purchase amount?
- Which age groups spend the most?
- Seasonal purchasing trends.
- Customer subscription analysis.

Used:
- Bar Charts
- Pie Charts
- Histograms
- Box Plots
- Correlation Analysis

### 4. SQL Analysis
Created SQL tables and executed queries such as:
- Total Sales
- Average Purchase Amount
- Top Product Categories
- Customer Demographics
- Revenue by Gender
- Monthly Sales Trends
- Customer Count by Location

### 5. Dashboard Development
Built an interactive Power BI dashboard including:
- KPI Cards
- Sales Trend Charts
- Customer Demographics
- Product Category Analysis
- Purchase Frequency
- Payment Method Distribution
- Filters and Slicers

### 6. Report & Presentation
Prepared:
- Project Report
- Power BI Dashboard
- PowerPoint Presentation summarizing business insights

## ▶️ How to Run

### Python
1. Clone this repository.
2. Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open the Jupyter Notebook.

```bash
jupyter notebook
```

4. Run all cells to perform data cleaning and analysis.

---

### SQL

1. Create a database in MySQL Server.
2. Import the dataset.
3. Execute the SQL queries from the **SQL** folder.

---

### Power BI

1. Open the `.pbix` file.
2. Refresh the dataset.
3. Explore the interactive dashboard.
